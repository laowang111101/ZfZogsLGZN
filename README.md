# 图书信息管理系统 java1062

## 项目概述

图书信息管理系统是基于Spring Boot框架开发的一个后台管理系统。该系统整合了Spring MVC和Mybatis框架，前端采用HTML、CSS、ElementUI、Vue.js及Ajax技术进行构建。系统遵循MVC设计模式，后端数据库采用MySQL。

## 技术栈

### 后端

- Spring Boot
- Spring MVC
- Mybatis
- MySQL

### 前端

- HTML
- CSS
- ElementUI
- Vue.js
- Ajax

## 功能模块

系统主要包括以下功能模块：

1. 用户认证
   - 登录
   - 注册
2. 图书管理
   - 添加图书
   - 删除图书
   - 修改图书信息
   - 查询图书

## 系统角色

- 系统管理员：负责维护整个系统的正常运行，对图书信息进行管理。

## 运行环境

- 服务器：推荐使用Linux系统
- 数据库：MySQL 5.7+
- IDE：IntelliJ IDEA 或 Eclipse

## 部署步骤

1. 准备运行环境：安装JDK、MySQL以及所需的服务器。
2. 克隆项目代码到本地。
3. 导入数据库脚本，创建所需的数据表。
4. 配置项目中的数据库连接信息。
5. 编译并运行项目。
